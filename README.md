# EventFlow – Event Booking Web Application

## Description

EventFlow is a web application created to make event booking easier for students and small organizations. Instead of using emails or manual sign-up sheets, this system allows users to view and book events online in a simple and convenient way.

Users can browse different events such as workshops, seminars, and networking sessions. They can also search for specific events and see details like date, location, and available seats. Once they find an event they are interested in, they can book it by entering their name and email.

The project focuses on building a working full-stack application. The frontend is designed using HTML, CSS, and JavaScript, while the backend is developed with Node.js and Express. Airtable is used to store event and booking data.

## Technologies Used

* HTML, CSS, JavaScript (Frontend)
* Node.js and Express (Backend)
* Airtable (Database)

## Features

* View list of events
* Search/filter events
* Book an event
* Responsive design
* Data stored in Airtable

## How to Run

### Backend

cd backend
npm install
node server.js

### Frontend

Open index.html using Live Server
